A joyful and engaging web application for practicing mathematics from Class 1 to Class 12 (ages 7-99).

## Features

- **10 Exercises Per Set**: Each session includes 10 carefully generated math problems
- **12 Class Levels**: From basic addition (Class 1) to advanced algebra and trigonometry (Class 12)
- **Kooklot System**: Earn Kooklot by completing all exercises correctly
- **Joyful Design**: Beautiful gradient colors and smooth animations
- **Sweet Sounds**: Pleasant audio feedback for correct answers, errors, and celebrations
- **Progress Tracking**: Your Kooklot are saved locally and persist between sessions

## How to Use

1. Open `index.html` in any modern web browser
2. Select your class level from the dropdown menu
3. Click "Start New Exercise Set" to begin
4. Enter your answers and click "Check" (or press Enter)
5. Complete all 10 exercises to see your results
6. Earn Kooklot by getting all answers correct!

## Class Levels

- **Class 1-2**: Simple addition and subtraction (1-20)
- **Class 3-4**: Addition, subtraction, simple multiplication (1-100)
- **Class 5-6**: All operations, larger numbers, simple division
- **Class 7-8**: Fractions, decimals, negative numbers
- **Class 9-10**: Algebra, powers, square roots
- **Class 11-12**: Advanced algebra, trigonometry, logarithms

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- Web Audio API (for sound effects)
- LocalStorage (for saving progress)

## Browser Compatibility

Works on all modern browsers that support:
- ES6 JavaScript
- CSS3 animations
- Web Audio API
- LocalStorage

## Installation

Simply download or clone this repository and open `index.html` in your web browser. No build process or dependencies required!
